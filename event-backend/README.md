

This is an event management project backend which has the following prerequisites:


1.   Node.JS + Express.JS for handling Admin and User requests.

2.  npm to install and manage packages in Node.JS. Some of the packages incude Express.JS(to create servers), jsonwebtoken(create JWT tokens), bcrypt(password            Hashing).

3.  My sql workbench

    For running backend use command in terminal - npm run dev.
