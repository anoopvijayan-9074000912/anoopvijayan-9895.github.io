This is an event management project which has been developed with front end flutter 
 
In pubspec.yaml file please include

1.   Dependencies
     flutter:
     sdk: Flutter

2.   Dev_dependencies:
            flutter test
            sdk: flutter

3.   State Management - provider

4.   HTTP client for API calls - dio

5.   Material Icons
     flutter_svg


     For running frontend use command in terminal -   flutter run -d chrome
